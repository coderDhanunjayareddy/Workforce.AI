# Workforce AI Typography Guide

Workforce AI uses a precise enterprise type system built around Sora for confident headings and Manrope for readable product UI.

## Font Families

| Role | Family | Usage |
| --- | --- | --- |
| Heading | Sora | Page titles, dashboard section titles, marketing headlines |
| Body | Manrope | Interface text, labels, paragraphs, tables, forms |
| Monospace | JetBrains Mono | IDs, code snippets, API examples, technical metadata |

## Font Weights

| Family | Approved Weights |
| --- | --- |
| Sora | 600, 700, 800 |
| Manrope | 400, 500, 600 |
| JetBrains Mono | Regular project monospace weight |

## Heading Scale

| Level | Size | Weight | Line Height | Usage |
| --- | --- | --- | --- | --- |
| H1 | 48px | 800 | 1.08 | Marketing hero and major presentation titles |
| H2 | 36px | 700 | 1.15 | Landing sections and major page groups |
| H3 | 28px | 700 | 1.2 | Dashboard groups, feature sections |
| H4 | 22px | 600 | 1.3 | Cards, panels, modal titles |
| H5 | 18px | 600 | 1.35 | Compact widget titles |
| H6 | 16px | 600 | 1.4 | Table groups and dense product surfaces |

## Rules

Keep letter spacing at `0`. Do not scale type fluidly with viewport width. Use tighter, smaller headings inside dense product panels and reserve large display type for true hero or presentation moments.
