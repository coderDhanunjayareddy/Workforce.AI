# Workforce AI Color Palette

The Workforce AI palette is designed for enterprise clarity: deep trust colors, crisp neutral surfaces, and restrained semantic accents.

## Primary Colors

| Token | Hex | Usage |
| --- | --- | --- |
| Primary Navy | `#0F1B3D` | Logo foundation, primary navigation, enterprise emphasis |
| AI Teal | `#14B8A6` | AI signal, active states, high-value accent moments |
| Secondary Blue | `#2563EB` | Links, focused actions, informational highlights |

## Backgrounds

| Token | Hex | Usage |
| --- | --- | --- |
| Background | `#F8FAFC` | App and documentation page background |
| Surface | `#FFFFFF` | Cards, modals, panels, tables |
| Dark Background | `#0B1220` | Dark mode shells, investor/demo presentation backgrounds |

## Typography Colors

| Token | Hex | Usage |
| --- | --- | --- |
| Primary Text | `#0F172A` | Headings, body copy, key labels |
| Secondary Text | `#64748B` | Supporting copy, metadata, helper text |

## Semantic Colors

| Token | Hex | Usage |
| --- | --- | --- |
| Success | `#10B981` | Healthy employee states, completed workflows |
| Warning | `#F59E0B` | Attention states, expiring knowledge, partial sync |
| Danger | `#EF4444` | Failed actions, destructive states, critical alerts |
| Info | `#3B82F6` | Neutral guidance, updates, system notices |

## Usage Examples

| Scenario | Recommended Color Treatment |
| --- | --- |
| Primary CTA | Primary Navy background with white text; teal icon or focus ring when useful |
| AI activity indicator | AI Teal dot or ring; avoid large teal fills |
| Dashboard KPI | Primary Text value, Secondary Text label, semantic trend color |
| Dark mode header | Dark Background with white text and AI Teal active state |
| Alert banners | Semantic color icon with restrained tinted background |

## Accessibility

Use Primary Navy or Primary Text for text-heavy surfaces. Reserve AI Teal for accent and signal use, especially when paired with sufficient contrast. Do not use semantic colors as the only indicator of meaning.
