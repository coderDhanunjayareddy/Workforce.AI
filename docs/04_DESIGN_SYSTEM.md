# PURPOSE

This document defines the complete visual language of Workforce AI.

Every page, module, component and interaction MUST use this design system.

Never invent new visual styles.

Consistency is more important than creativity.

==============================================================================
1. DESIGN PHILOSOPHY
==============================================================================

The application should feel like:

• Enterprise
• Premium
• Trustworthy
• Executive
• Calm
• Intelligent
• Minimal
• Modern

Imagine if:

Stripe
OpenAI
Notion
HubSpot
Linear

designed one enterprise platform together.

Avoid:

❌ Neon colors

❌ Gaming UI

❌ Heavy gradients

❌ Glassmorphism

❌ Fancy animations

❌ Oversized cards

Everything should communicate trust.
